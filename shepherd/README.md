# Shepherd — Pastoral Network App

A church pastoral network management system with Google Sheets integration and role-based Google login.

## Project Structure

```
shepherd-app/
├── index.html
├── package.json
├── vite.config.js
├── netlify.toml
└── src/
    ├── main.jsx
    └── App.jsx
```

## Deploying to Netlify

1. Upload all these files to a GitHub repository
2. Connect the repository to Netlify
3. Netlify will auto-build and deploy

## First-Time App Setup

Once deployed, open your app URL and follow the in-app setup guide to connect Google Sheets and Google OAuth.
